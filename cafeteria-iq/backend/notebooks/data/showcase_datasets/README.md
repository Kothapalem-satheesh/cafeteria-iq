# Showcase Datasets

Use these CSV files in class demos to showcase how clustering and recommendations change by behavior pattern.

## Files
- `01_balanced_weekly_flow.csv`: Balanced demand across all slots and categories.
- `02_lunch_rush_corporate.csv`: Strong lunch rush with higher average order values.
- `03_evening_snackers_students.csv`: Student-heavy behavior with snacks and beverages.
- `04_premium_weekend_family.csv`: Family weekend spikes with premium basket sizes.
- `05_health_conscious_low_calorie.csv`: Lower spend, lighter menu choices, fewer fried items.
- `06_anomaly_festival_week.csv`: Festival week with sudden spikes and unusual bundles.

Each file has 1000 rows and follows the same schema as `transactions.csv`.
