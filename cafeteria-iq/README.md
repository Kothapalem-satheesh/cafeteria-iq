# Cafeteria IQ (application)

All source code for the app lives in this directory.

**Documentation:** the full setup guide, architecture diagram, and API notes are in the [**README at the repository root**](../README.md).

Quick start from the repo root:

```bash
cd cafeteria-iq/backend
cp .env.example .env
npm install
npm run dev
```

Use paths under `cafeteria-iq/` as described in the root README.
